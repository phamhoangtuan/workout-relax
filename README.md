# Office Relax — 30 Min Daily Workout

Static web app for office workers. 30-minute guided workout to counter the effects of sitting 8+ hours a day.

**[▶ Open App](https://phamhoangtuan.github.io/workout-relax/)**

## What it does

- **2 alternating routines** (A: Mon/Wed/Fri, B: Tue/Thu/Sat) — different exercises, same muscle targets
- **4 phases per session**: Warm-Up → Strengthening (3 circuits) → Stretching → Neural Reset
- **Countdown timer** per exercise, auto-advances through 26 exercises
- **Progress tracking** — per-exercise timer bar + total session progress
- **Audio beeps** on exercise/phase transitions
- **Keyboard shortcuts**: `Space` pause, `→` skip
- **Sunday**: rest day with recovery message

## Target muscles

| Tight (stretch) | Weak (strengthen) |
|---|---|
| Hip flexors, Hamstrings | Glutes (maximus + medius) |
| Pecs, Upper traps | Deep core, Lower traps |
| Calves, Suboccipitals | Deep neck flexors |

Addresses both Upper and Lower Crossed Syndrome from prolonged sitting.

## Tech

HTML + CSS + vanilla JS. Single-page app, no dependencies, no build step. Works offline from `file://` or deployed on GitHub Pages.
